# SOR4 Mod Launcher

An unofficial, fan-made mod launcher for **Streets of Rage 4** that lets you
switch between mod files instantly — no manual file replacement needed.

---

## How It Works

Streets of Rage 4 loads its game data from a single file called `bigfile`.
This launcher creates a symbolic link (symlink) named `bigfile` that points to
whichever file you select — the original or a mod. Switching is instant and
non-destructive: your original game file is never modified or deleted.

---

## Features

- Switch between up to 5 mods from a clean menu
- Launches the game directly via Steam after selecting a mod
- Custom display names for each mod
- One-click restore to the original game files
- Settings screen to configure game path and mods folder

---

## Installation

### Executable (recommended)

1. Download and extract `SOR4ModLauncher.zip`
2. Place your mod files inside the `/mods` folder
3. Right-click `SOR4ModLauncher.exe` → **Run as Administrator**
   > Symlinks on Windows require administrator privileges
4. On first launch, go to **Settings** and set your game installation path
5. Enable the mods you want and click **Save**

### From source

**Requirements:** Python 3.11+

```bash
pip install -r requirements.txt
python main.py
```

> Run as Administrator — required for symlink creation on Windows.

## Usage

| Action | Result |
|---|---|
| Click a mod name | Activates the mod and launches the game |
| Click **Original** | Restores original bigfile and launches the game |
| Click **Restore original** | Restores original files and clears mod list |
| Click **Settings** | Opens settings screen |
